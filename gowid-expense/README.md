# gowid-expense

Gowid 법인카드 경비를 Claude Code / Codex에서 자연어로 조회·제출하는 스킬.

> "내 미제출 경비 보여줘" → 조회 → "1번 식비로 제출해" → 완료

## 설치

```bash
# Claude Code
npx skills add EO-Studio-Dev/gowid-expense-skill --skill gowid-expense --agent claude-code -y

# Codex는 설치 후 ~/.codex/skills/gowid-expense 에 연결되도록 배치
```

API 키가 내장되어 있어서 설치만 하면 바로 사용 가능합니다.

## 사용법

Claude Code 또는 Codex에서 자연어로 요청하면 됩니다.

**경비 조회**
```
"내 경비 보여줘"
"미제출 경비"
"이번 달 경비 내역"
```

**경비 제출 (일반)**
```
"1번 IT서비스로 제출해"                    → 자동 분류 규칙에 매칭되는 항목
"32625805 도서구입비로 제출해"              → expense ID 직접 지정
"Notion 결제 건 IT서비스로 제출해"          → 가맹점명으로 찾기
```

**식비 제출 (참석자 필요)**
```
"3번 점심식비로 제출해, 김찬희랑 먹었어"    → 참석자 자동 매칭
"야근식비로 제출해"                        → 19시 이후 결제분
"금요미식회로 제출 — 팀 전체"              → 인당 15,000원 한도
```

**규칙 검색**
```
"자동규칙 검색 Notion"                     → Notion 관련 분류 규칙 확인
"zoom 규칙"                               → Zoom 관련 규칙
```

**기타**
```
"용도 목록"                               → 58개 전체 카테고리
"팀원 목록"                               → 식비 참석자 선택용
"경비 상세 32625805"                      → 특정 건 상세 정보
```

## 작동 방식

1. `git config user.email`로 본인 식별 (Gowid 계정과 자동 매칭)
2. Gowid API에서 본인 카드의 미제출 경비만 조회
3. 289개 자동 분류 규칙으로 용도 추천
4. Claude가 대화형으로 제출 처리 (메모, 참석자 등)
5. 이미 제출된 건은 자동으로 건너뜀 (중복 방지)

## 지원 기능

- **경비 조회**: 본인 카드의 미제출 내역
- **경비 제출**: 용도 지정 + 메모 + 참석자 (식비)
- **상세 조회**: 카드번호, 승인일, 금액, 현재 상태
- **용도 목록**: 58개 활성 카테고리
- **팀원 목록**: 참석자 선택용
- **자동규칙**: 가맹점별 자동 분류 규칙 검색 (289개)
- **규칙 제안**: GitHub Issue로 새 규칙 제안

## 자주 쓰는 용도

**식비**

| 용도 | 한도 | 필요 정보 |
|------|------|-----------|
| 점심식비 | 인당 12,000원 | 초과 시 참석자 필수 |
| 야근식비 | 인당 12,000원 | 19시 이후 사용분 |
| 금요미식회 | 인당 15,000원 | 금요일 점심, 참석자 필수 |
| 회식비 | 없음 | 참석자 필수 |

**SaaS / 구독**

| 용도 | 예시 | 필요 정보 |
|------|------|-----------|
| IT서비스 이용료 | Notion, Slack, Zoom, GitHub, Figma, AWS | 메모에 서비스명 |
| 멤버십 구독료 | WeWork, 공유오피스, ChatGPT | 메모에 서비스명 |
| 매거진 구독료 | 뉴닉, Economist, 뉴스레터 | 메모에 매체명 |

**교통 / 출장**

| 용도 | 필요 정보 |
|------|-----------|
| 업무교통비 | 출발지/도착지 (택시, 버스, 지하철) |
| 야근교통비 | 22시 이후 귀가 택시 |
| 주차비 | - |

**기타**

| 용도 | 필요 정보 |
|------|-----------|
| 도서구입비 | - |
| 소모품비(10만원이하) | 사무용품, 어댑터 등 |
| 우편비 | 택배, 등기 |
| 업무추진비 | 촬영 스탭식비/음료 등, 프로젝트 지정 필수 |

## API 제출 제한

일부 용도는 Gowid 서버에서 필수항목을 강제하여 API로 제출이 안 됩니다.
해당 건은 Gowid 웹에서 직접 제출해주세요.

| 용도 | API 제출 | 비고 |
|------|---------|------|
| IT서비스, 식비, 회식비, 우편비, 통신비 등 | 가능 | 대부분의 용도 |
| 업무교통비, 야근교통비 | 불가 | 출퇴근 시간/출발지 필수 |
| 도서구입비 | 불가 | 책 제목 필수 |
| 서류발급비 | 불가 | 프로젝트명 필수 |

## 요구사항

- [Claude Code](https://claude.ai/claude-code) 설치
- Python 3.9+
- Gowid 법인카드 계정 (git email과 Gowid 이메일 일치 필요)
- git email이 다른 경우: Claude Code settings에 `"env": { "GOWID_EMAIL": "본인@eoeoeo.net" }` 추가
