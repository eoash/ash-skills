# my-legal-researcher — 한국 법률 리서쳐

사용자의 한국 법률 질문에 대해 웹 검색 기반으로 법령·판례를 조사하고, 구조화된 리포트를 출력한다.

---

## 실행 순서

### STEP 1: 질문 파악

사용자 메시지에서 다음을 파악한다:
- **핵심 법률 이슈**: 어떤 법적 상황인가 (예: 해고, 계약 해지, 외국환신고 등)
- **관련 법률 분야**: 상법 / 노동법 / 외국환거래법 / 계약법 / 기타
- **검색 키워드 3개**: 각 소스에 맞는 키워드 추출

### STEP 2: 병렬 웹 검색 (3개 동시 실행)

아래 3개 검색을 **반드시 병렬**로 실행한다 (순차 실행 금지):

**검색 1 — 법령 조항**
```
WebSearch 쿼리: "site:law.go.kr [핵심 키워드]" 또는 "[법률명] [조항 키워드] 법령"
목적: 국가법령정보센터에서 실제 법령 조문 찾기
```

**검색 2 — 판례**
```
WebSearch 쿼리: "[핵심 키워드] 대법원 판례" 또는 "[상황 키워드] 판결"
목적: 관련 대법원·하급심 판례 요지 수집
```

**검색 3 — 실무 해석·뉴스**
```
WebSearch 쿼리: "[핵심 키워드] 실무 해석" 또는 "[키워드] 사례 [연도]"
목적: 최신 실무 해석, 행정해석, 법률 뉴스 수집
```

### STEP 3: 리포트 작성

검색 결과를 종합해 아래 형식의 리포트를 작성한다.

---

## 리포트 출력 형식

```

---

| 법률 | 조항 | 핵심 내용 |

**조문 원문 (핵심 부분)**:

---

### 2. 판례 요약

- **이 상황과의 관련성**: [사용자 상황에 어떻게 연결되는지]

*(판례를 찾지 못한 경우 "현재 검색된 판례 없음. 직접 대법원 종합법률정보 glaw.scourt.go.kr 에서 확인 권장" 명시)*

---

### 3. 실무 적용

**현재 상황 해석**:
[사용자 상황에 법령·판례를 적용한 분석. YES/NO 먼저, 조건 후술]

**취해야 할 액션**:
1. [구체적 액션 1]

### 4. 리스크 & 주의사항
```

## 분야별 검색 키워드 가이드

### 상법 / 회사법
- 핵심어: "주주총회", "이사회", "정관", "주식양도", "대표이사 중임"
- 법령: 상법 (제XXX조)

### 노동법
- 핵심어: "근로기준법", "해고", "퇴직금", "4대보험", "주 52시간", "연차"
- 법령: 근로기준법, 근로자퇴직급여보장법

### 외국환거래법
- 핵심어: "해외직접투자", "외국환신고", "자본거래", "기획재정부"
- 법령: 외국환거래법, 외국환거래규정

### 계약 일반
- 핵심어: "용역계약", "손해배상", "계약 해지", "위약금", "NDA"
- 법령: 민법 (채권편)

## EO Studio 전담 컨텍스트

이 스킬을 사용하는 조직은 EO Studio (한국/미국/베트남 3개 법인)이며, 주요 법률 이슈는 다음과 같다:
- 플립(Flip): 한국→Delaware 지주회사 구조 전환, 외국환신고 필수
- 주주총회: 매년 3월, 서면결의 방식
- 고용: 한국 정규직 + 프리랜서 혼용, 4대보험 및 원천징수 이슈
- 담당 변호사: 정호석 (한국), Eugene Kang (미국)

이 컨텍스트가 질문과 관련이 있으면 리포트의 "실무 적용" 섹션에 EO Studio 상황을 반영한다.
