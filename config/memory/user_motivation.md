## Claude Code 사용 철학 (32일차 회고, 2026-03-13)

- **동기**: 전사적 AI 도입을 위해 "나부터 잘 써보자"로 시작. 리더십 원칙은 솔선수범
- **핵심 깨달음**: 쓰기 전엔 "중요하겠다" 수준 → 쓰고 나니 "안쓰면 큰일난다"는 절박함
- **자산화 원칙**: "툴에서 하면 액션이 휘발되지만 클로드에서 하면 자산이 된다" — Slack/Gmail/Calendar도 가능하면 Claude를 통해 진행
- **학습 방식**: 습작보다 실제 프로젝트(token-dashboard 등)를 깊게 파면서 배움. 떠오르는 아이디어는 바로 구현
- **현재 심경**: 딸깍(FOMO)이 아닌 순수한 재미와 호기심. 즐기면서 하게 됨
- **비전**: 팀원 전원의 세컨드브레인 연결 → "프로토스 칼라" 같은 시너지
- **비개발자 강점**: "이거 되면 안 되나?"에서 출발하는 상상력. 기술적 프레임 없이 사용자 관점에서 자동화 설계
- **전파 활동**: 사내 세션 2회 + 베트남 팀 세션 + 1대1 현장 교육 진행 중
